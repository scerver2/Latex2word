🌌 CONVERTIDOR DE LATEX A WORD 🌌



Una eina web moderna i ràpida per renderitzar documents acadèmics escrits en LaTeX i exportar-los fàcilment a format Word (.doc) o copiar-los directament al porta-retalls.



✨ CARACTERÍSTIQUES PRINCIPALS



🔒 Processament al Navegador: Tota la conversió es fa localment al teu ordinador. No s'envien dades a cap servidor extern, garantint la teva privacitat al 100%.



🧮 Renderització de Fórmules: Integració perfecta amb MathJax per compilar equacions matemàtiques i científiques complexes en temps real.



📄 Exportació a Word: Generació d'arxius .doc amb estil acadèmic clàssic (font Times New Roman, interlineat adequat, i estructures jeràrquiques).



📋 Còpia Intel·ligent: Botó de copiar dissenyat per mantenir el format enriquit (Rich Text). Enganxa'l directament a Word, Google Docs o altres editors d'ofimàtica.



🎨 Interfície Moderna: Disseny tipus Dark Mode / Cyberpunk amb tipografies especialitzades (JetBrains Mono per al codi, Crimson Pro per als textos) per a la millor usabilitat.



🛠️ COMANDES LATEX SUPORTADES



El convertidor està dissenyat per interpretar l'estructura bàsica d'un document acadèmic o un full de resolució d'exercicis:



1\. Estructura i Text



\\section{...}, \\subsection{...}, \\subsubsection{...} ➡️ Capçaleres i títols.



\\textbf{...} ➡️ Text en negreta (suporta claus niuades per fórmules internes).



\\textit{...} ➡️ Text en cursiva.



{\\Large ...} ➡️ Text de mida més gran.



\\begin{center} ... \\end{center} ➡️ Centrar blocs de text.



\\begin{itemize} \\item ... \\end{itemize} ➡️ Llistes de punts clàssiques.



2\. Espaiat i Format



\\vspace{...} i \\\\ ➡️ Salts de línia / línies en blanc.



\\hrule ➡️ Línia horitzontal separadora.



\\noindent ➡️ Evita la sagnia (processat automàticament).



3\. Matemàtiques (via MathJax)



Fórmules en línia: Utilitza $ ... $ o \\( ... \\)



Fórmules en bloc (centrades): Utilitza $$...$$ o \\\[ ... \\]



🚀 COM UTILITZAR-LO



📝 Enganxa el teu codi: Introdueix el codi font LaTeX a la columna esquerra. No passa res si hi inclous el preàmbul complet (\\documentclass, \\usepackage), el convertidor és prou llest per extreure'n només el contingut útil.



⚙️ Processa el document: Fes clic al botó verd "Renderitzar Fórmules".



👀 Revisa la previsualització: A la columna dreta apareixerà un "foli blanc" mostrant el resultat final amb totes les equacions processades.



💾 Exporta el resultat:



Fes clic a "Copiar" si vols enganxar-ho tu mateix al teu document de referència.



Fes clic a "Descarregar .doc" per obtenir directament un arxiu llest per obrir amb Microsoft Word.
