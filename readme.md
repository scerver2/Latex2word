Convertidor de LaTeX a WordUna eina web moderna i ràpida per renderitzar documents acadèmics escrits en LaTeX i exportar-los fàcilment a format Word (.doc) o copiar-los directament al porta-retalls.✨ CaracterístiquesProcessament al Navegador: Tota la conversió es fa localment al teu navegador. No s'envien dades a cap servidor, garantint total privacitat.Renderització de Fórmules: Integració amb MathJax per compilar equacions matemàtiques complexes en temps real.Exportació a Word: Generació d'arxius .doc amb estil acadèmic (Times New Roman, interlineat adequat, estructures jeràrquiques).Còpia Intel·ligent: Botó de copiar que manté el format enriquit (Rich Text) perquè el puguis enganxar directament a Word, Google Docs o altres editors.Interfície Moderna: Disseny tipus Dark Mode / Cyberpunk amb tipografies especialitzades (JetBrains Mono per al codi, Crimson Pro) i alta usabilitat.📝 Comandes LaTeX SuportadesEl convertidor està dissenyat per interpretar l'estructura bàsica d'un document acadèmic o de resolució d'exercicis.Estructura i Text\section{...}, \subsection{...}, \subsubsection{...}: Capçaleres i títols.\textbf{...}: Text en negreta (suporta claus niuades).\textit{...}: Text en cursiva.{\Large ...}: Text de mida gran.\begin{center} ... \end{center}: Centrar el text.\begin{itemize} \item ... \end{itemize}: Llistes de punts.Espaiat i Format\vspace{...} i \\: Salts de línia / línies en blanc.\hrule: Línia horitzontal separadora.\noindent: Evita la sagnia (processat automàticament).Matemàtiques (via MathJax)Formules en línia: $ ... $ o \( ... \)Formules en bloc: $$...$$ o \[ ... \]📖 Com utilitzar-loEnganxa el teu codi: Introdueix el codi font LaTeX a la columna esquerra. No passa res si inclous el preàmbul (\documentclass, \usepackage), el convertidor extreurà automàticament el contingut de dins del \begin{document} ... \end{document}.Processa el document: Fes clic al botó verd "Renderitzar Fórmules".Revisa la previsualització: A la columna dreta apareixerà un "foli blanc" mostrant el resultat final amb totes les equacions resoltes.Exporta:Fes clic a "Copiar" si vols enganxar-ho manualment al teu document de Word ocreant un nou document de Google Docs.Fes clic a "Descarregar .doc" per obtenir directament un arxiu obrible amb Microsoft Word.🌐 CompatibilitatL'eina web funciona en qualsevol navegador modern amb suport per a JavaScript avançat:Chrome / Chromium Edge: ✅ Totalment compatibleFirefox: ✅ Totalment compatibleSafari: ✅ Totalment compatibleOpera / Brave: ✅ Totalment compatible🛠 Instal·lació i Ús LocalCom que l'eina és 100% frontend (HTML, CSS, JS), no necessites instal·lar cap servidor.Descarrega o guarda el fitxer convertidor.html al teu ordinador.Fes doble clic per obrir-lo amb el teu navegador web preferit.Ja pots utilitzar-lo (necessitaràs connexió a internet la primera vegada perquè carregui MathJax, Tailwind CSS i les fonts de Google).👨‍💻 Autor i CrèditsCreat per Salvador Cervera per a la comunitat amb ❤️.Aquest projecte utilitza:Tailwind CSS per la maquetació i disseny.MathJax per la renderització tipogràfica de fórmules matemàtiques en alta qualitat.Google Fonts (JetBrains Mono & Crimson Pro).
